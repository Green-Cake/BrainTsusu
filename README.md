# BrainTsusu
「つ」「す」「う」の三文字だけでProgrammingができちゃう！

# Strong Point
伝言ができる！(簡単にとは言ってない)

# How to write
Brainf\*ukの各記号を置き換えただけです。
詳しくはそちらをば。
対応表:
```
\> つつ
\< つす
\+ つう
\- すつ
\. すす
\, すう
\[ うつ
\] うす

うう => 数字として出力(オリジナル)
```
# How to use
上のBrainTsusu.jarをDownload.ターミナルからjarのあるとこまでcdして実行してください。

ファイルを指定する場合、 `-file FILE_PATH`　で。
